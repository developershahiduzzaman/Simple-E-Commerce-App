## üöÄ 7-Day Roadmap (Project Progress)
* **Day 4**: Initial Android project setup, built Login/Register UI, and integrated APIs using Retrofit.
* **Day 4**: Implemented Product listing with RecyclerView to display items from the backend dynamically.

* **Day 5**: Integrated the Place Order API and created a dedicated Order History screen.
* **Day 6**: Integrated Online Payment Gateway (SSLCommerz/Stripe) in test mode with success/failure handling.

## ‚ú® Key Features
* **Secure Authentication**: Token-based login and registration using Laravel Sanctum for protected API access.
* **Dynamic Product Catalog**: Browse products by categories with a smooth and responsive RecyclerView interface.
* **Cart System**: A fully functional shopping cart where users can add items and update quantities easily.
* **Secure Checkout**: Complete end-to-end purchase flow with secure payment gateway integration.
* **Order Tracking**: Users can view their previous orders and transaction details in the Order History screen.
